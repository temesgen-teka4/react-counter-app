®️ React Counter App

A React-based counter application with increment, decrement, and reset functionality. Displays a warning message when the maximum value is reached.

-> Features
- Increment and decrement buttons with limits
- Reset button to restart the counter
- Warning message when count reaches 10
- Styled with CSS for a modern UI

-> Tech Stack
- React (Hooks: useState, useEffect)
- CSS
